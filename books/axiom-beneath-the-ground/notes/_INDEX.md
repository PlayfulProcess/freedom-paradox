## How to add a note

1. **Filename pattern:** `<topic-slug>-YYYY-MM-DD.md` (e.g. `dwarkesh-prize-DRAFT-A-responsibility-infrastructure-2026-04-26.md`).
2. **Frontmatter (top of file):**
   ```
   ---
   status: draft | active | archived | superseded
   topic: <one-line description>
   created: YYYY-MM-DD
   last-edit: YYYY-MM-DD
   serves: <which book / chapter / external piece this informs>
   related: <list of related files in notes/ or research/>
   ---
   ```
3. **Update this index** with one line: status icon + filename + ≤140 chars description.
4. **At the end of a working session:** scan this index for stale entries; demote 🟢 → 🟡 if the work is parked, ⚪ if superseded.

## How to retire a note

- If superseded by a newer file: change status to ⚪, add `(superseded by <newer-file>)` to its line in this index. **Do not delete** — Draft Preservation rule (CLAUDE.md).
- If abandoned: change status to ⚪ with a note explaining why.
