# Deep Research Prompts — Claude.ai (sequential)

This file contains six standalone prompts to paste into Claude.ai, one at a time, with Research (or Deep Research) turned on. Each prompt is self-contained — Claude.ai does not have access to this repo, so each prompt carries its own context.

## How to use

1. Open claude.ai.
2. Turn on **Research** (or **Deep Research** if available) before sending.
3. Copy **Prompt 1** below, paste it into a new conversation, send.
4. When Claude finishes, copy the full output into a new file locally at the path specified in the prompt.
5. Start a **new conversation** for Prompt 2. Repeat for Prompts 3–6.
6. When all six files are saved, tell me and I will draft Ch1.

Starting a new conversation for each prompt keeps Claude.ai's research focused and prevents context drift between unrelated topics.

## Copyright rule (applies to every prompt)

No substantial verbatim passages from copyrighted works in any saved file. Paraphrase is the default. Brief fair-use quotations (≤30 words) are permitted only where the exact wording is argumentatively necessary, and must be attributed with full citation (author, title, edition, year, page). Never fabricate page numbers, quotes, or citations. If a source cannot be confirmed, use a `[RESEARCH NEEDED: specific question]` flag.
