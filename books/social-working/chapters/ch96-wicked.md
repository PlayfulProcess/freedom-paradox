# Wicked

## Reflection

What if the villain was right all along—and what's lost when we make that message palatable?

## Description

The sanitized merger (1995/2003). Wicked shifts race → color (green), making oppression 'digestible' for wider audiences. The Wiz was explicit about Blackness; Wicked encodes the same othering but in a way that lets white audiences feel comfortable. 'No one is born wicked'—but also, no one has to confront actual racism.

## Key Insight

Wicked is the colonial compromise: keeps the message about vilification of the 'other,' but removes the specificity that would make white audiences uncomfortable. Green is safer than Black.
