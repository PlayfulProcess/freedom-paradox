# Social Working

## Reflection

What if everyone and everything is always already social working?

## Source

Analepsis: from Chapman & Withers' call for 'a persistent dredging operation' of history. The mainstream has never run clean, perhaps never can.

## Description

Not social work (noun, profession, colonial)—social working (verb, what all beings do). Closing analysis in 'social work as profession' is already colonizing. Decolonizing means putting social work in a social working perspective. This deck is an analeptic exploration: holding contradictions without resolving them, dredging the mainstream that has never run clean.
