# Raw Research: John O'Nolan — "Open Source in the age of AI"

**Source:** Ghost newsletter by John O'Nolan
**Date:** February 26, 2026
**Relevance:** Chapter 2, Chapter 16 (closing)

## Key Arguments

1. **AI is overturning the premise that code is scarce.** Open source licenses were built on the assumption that code is intrinsically valuable because it's difficult to maintain and expensive to write. AI collapses this.

2. **The Cloudflare/Vercel incident.** Cloudflare took a competitor's open-source codebase and had AI rewrite it from scratch in one week. "If anyone can point Claude at an open source codebase and have it rewritten from scratch, without actually using any of the original code, what does that mean for software licenses?"

3. **The trajectory applies to closed source too.** "I don't imagine it will be long before any sufficiently successful proprietary product with a public-facing interface can be reverse-engineered and rebuilt by a motivated competitor with access to frontier models."

4. **The paradox.** "When Stallman launched the free software movement in 1983, it was a reaction to a world where software was closed, proprietary, and wielded as an instrument of control... I'm beginning to wonder if AI might end up fulfilling that vision more completely than open source ever did."

## Key Quotes

> "Any open source project that achieves a degree of success will at some point face the reality of a competing company taking your open source code and using it to compete with you."

> "The dictating of the rules has largely been built upon a premise that code is intrinsically valuable because it's a scarce resource. Difficult to maintain. Expensive to write. AI is now rapidly overturning that premise."

> "If anyone can point Claude at an open source codebase and have it rewritten from scratch, without actually using any of the original code, what does that mean for software licenses? More specifically: Do software licenses mean anything?"

> "In a strange way, I'm beginning to wonder if AI might end up fulfilling that vision more completely than open source ever did."

## Context
- O'Nolan is founder of Ghost, an open-source nonprofit CMS
- Ghost has experienced Substack copying its source code
- He references Simon Willison (December 2025) and Geoffrey Huntley's reverse-engineering work
- Ghost is MIT-licensed, nonprofit, Kickstarter-funded — the model case of ethical open source
- Even HE is questioning whether the paradigm holds
