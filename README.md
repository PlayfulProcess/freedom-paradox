# PlayfulProcess — Books

A multi-book writing project by PlayfulProcess.

## Books

### 1. The Freedom Paradox: Open Source, AI, and the Limits of Openness
**Status: Complete first draft (~85K words)**

Open source was born as a freedom movement for benign software. AI is the first technology where "openness is always better" confronts genuine civilizational risk. 16 chapters + epilogue across 5 parts.

→ `books/freedom-paradox/`

### 2. Grammars of the Living World
**Status: Early draft + active research**

Contemplative and divinatory systems (I Ching, Ifá, Tarot, Kabbalah, astrology) as "grammars" — finite symbolic vocabularies that generate infinite meaning. Applies Kelty's recursive public framework to build a digital commons for these systems at recursive.eco.

→ `books/grammars-of-the-living-world/`
